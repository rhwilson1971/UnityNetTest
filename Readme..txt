This is a very bare bones project that show TCP networking data flow

Two projects must be build.  First on the NetworkManagerObject, check Is Server, then build the server files
Then, on the NetworkManagerObject, uncheck Is Server, the build thee client files

Be sure to build in separte folders

Run the server executable first, Test123.exe
Then, run the client executable, Test123.exe.

Input a message into the Input field on the client instance and Click the Send To Server button
On the server instance, type some text into the input field then click the Send to Client button

Messages will be displayed on the screen
